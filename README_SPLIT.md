# AIshopr Split Project

Diese Version teilt die bisherige Single-File-App in eine sauberere Struktur:

- `index.html` enthält HTML-Grundstruktur
- `css/styles.css` enthält das komplette Styling
- `js/app.js` enthält die komplette App-Logik

Die Funktionalität soll identisch zur vorherigen Fix-4-Version bleiben.
